# Quasar App (vue3-quasar-project)

### Install the dependencies

```ruby
npm i
```

### Start the app in development mode

```ruby
npm run dev
```


### 프레임워크
- [vue.js@3.3.4](https://ko.vuejs.org/guide/introduction)
- [quasar@2.12.0](https://quasar.dev/)
